v2: LangGraph 架构的重构方案。
这版的关键变化：
用 StateGraph 重写了调度层
每个 agent / tool 变成独立 graph node
用条件边处理：
渲染成功 → 上传 OSS
渲染失败且可重试 → 诊断 → 验证 → 修复 → 下一轮渲染
超过重试上限 → 失败收尾
保留了提示词文件化、任务归档、issue 验证、notice 学习

这套写法是按 LangGraph 官方推荐的 State + Nodes + Edges 模式来的，图里节点通过共享状态读写，条件分支用 add_conditional_edges 实现；安装上也按官方文档把 langgraph 放进了依赖，并保留了 langchain 作为生态兼容依赖。
